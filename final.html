Crash Detection System for CNC Machines using Vibration Analysis Techniques and Machine Learning Methods

Author: Trien Tri Nguyen
Supervisor: Dr Lahcen Ouarbya

extra

Overview

This project develops a real-time crash detection system for CNC machines using vibration data and machine learning. It utilizes an ADXL345 sensor and models such as GRUs and LightGBM to detect anomalies and prevent damage during machining operations.



Methodology

Vibration data is collected during machining, processed into frequency bins, and used to train predictive models. Predictions are compared against real-time signals to detect crashes.



Key Components

  • Sensor: ADXL345 Accelerometer
  • Machine: 3-axis desktop CNC 3018
  • Data Processing: FFT (Fast Fourier Transform)
  • Models: GRUs (for time-domain), LightGBM (for frequency-domain)

Results

Crash detection accuracy: Emergency stop triggered within 1.5 mm after impact

Challenges & Future Work

  • Low-resolution sensor limited detection accuracy
  • Real-time inference requires optimization
  • Potential improvements with AE sensors and acoustic fingerprinting


Conclusion

The system demonstrates that vibration analysis, combined with machine learning, can provide scalable and accurate real-time crash detection for CNC machines. Future work aims to extend it to a proactive crash prevention system.
